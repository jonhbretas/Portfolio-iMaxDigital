# Project Screenshots

Drop your project thumbnail images here.

**Recommended specs:**
- Size: 1280 x 800px (16:10)
- Format: .jpg or .webp
- Max ~200KB per image (compress with TinyPNG)

**Naming convention:** use lowercase, hyphenated names matching `projects.json`:
- `dental-clinic.jpg`
- `fashion-store.jpg`
- `law-firm.jpg`
- etc.
